# To view live

[On Heroku](http://vast-beach-39095.herokuapp.com/)

# To run the app

1. `git clone git@github.com:aaronfalloon/5-day-forecast.git`
2. `npm install`
3. `cd 5-day-forecast`
4. `npm run build`
5. `npm run serve`
6. Go to [http://localhost:3000/](http://localhost:3000/)

# To run the tests

`npm test`
